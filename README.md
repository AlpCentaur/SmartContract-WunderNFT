# Commands:

## Run Test suite:
```bash
npx hardhat test
```

## Deploy to test network:
Mumbai: 
```bash
npx hardhat run scripts/deploy.js --network mumbai
```

Rinkeby:
```bash
npx hardhat run scripts/deploy.js --network rinkeby
```

## Deploy to mainnet
```bash
npx hardhat run scripts/deploy.js --network polygon
```
